[Wireframe - Agência de Viagens
Este projeto é um protótipo visual desenvolvido com uma estética de rascunho feito à mão. O objetivo principal é validar a estrutura, o layout e o fluxo de conteúdo para o site de uma agência de turismo antes de avançar para a fase de design final.

O Design
A identidade visual foi pensada para simular um papel de rascunho:

Tipografia e Detalhes: Uso de fontes mais descontraídas e elementos com uma leve inclinação para transmitir a sensação de um desenho feito à mão.

Texturas e Cores: O fundo utiliza um tom suave que lembra pergaminho ou papel antigo, complementado por bordas tracejadas nos espaços reservados.

Estrutura: O layout conta com cabeçalho, banner principal com barra de pesquisa fictícia, vitrine de destinos, seção de diferenciais, depoimento de cliente e rodapé.

Tecnologias
HTML5 para a estruturação semântica.

CSS3 para o layout e estilização (Flexbox, sombras sólidas e bordas tracejadas).
